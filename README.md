# Tasks Organizer (Python)

Simple command-line task manager built in Python.

## Features

- Add tasks
- List tasks
- Mark tasks as compled
- Set task priority (1-5)
- Tasks saved in a JSON file

## Technologies

- Python
- JSON

## How to Run

1. Clone the repository
2. Run the program:
python main.py

## Example

1. List tasks
2. Add task
3. Complete task
4. Exit

Each task can have a priority from **1 ( very important)** to **5 (low priority)**.
